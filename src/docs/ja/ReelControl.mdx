# リール制御・目押し

`ReelController` は内部抽選の当選役に基づいて各リールの停止位置を決定します。引き込み（Slip）・蹴飛ばし（Reject）・AutoStop・取りこぼし判定を提供します。

## 基本的な使い方

```ts
import { ReelController } from 'reeljs';

const controller = new ReelController({
  reelStrips: [
    ['cherry', 'bell', 'bar', 'seven', 'replay', 'bell', 'watermelon'],
    ['bell', 'bar', 'seven', 'cherry', 'replay', 'bell', 'watermelon'],
    ['bar', 'seven', 'cherry', 'bell', 'replay', 'watermelon', 'bell'],
  ],
  slipRanges: [4, 4, 4],
  stopOrder: [0, 1, 2], // 左 → 中 → 右
});

// リール0の停止位置を決定
const result = controller.determineStopPosition(0, winningRole, 2);
console.log(result.actualPosition, result.isMiss);

// 停止コールバック登録
controller.onStop((result) => {
  console.log(`リール${result.reelIndex}: pos=${result.actualPosition}, miss=${result.isMiss}`);
});
```

## 引き込み（Slip）と蹴飛ばし（Reject）

**引き込み:** 対象シンボルが `SlipRange` 以内にある場合、リールが「引き込み」で対象位置に停止します。

**蹴飛ばし:** 当選役がMISSでないのに対象シンボルが `SlipRange` 外の場合、対象シンボルを避けるように停止位置をずらします（取りこぼし）。

```ts
// SlipRange = 4: 最大4コマ先まで引き込み可能
// stopTiming = 2, 'cherry' が位置5 → 3コマ引き込み → actualPosition = 5
// stopTiming = 2, 'cherry' が位置8 → 範囲外 → 蹴飛ばし → isMiss = true
```

## AutoStop

`autoStop` 有効時、全リールが `stopOrder` に従ってランダムタイミングで停止します。

```ts
const controller = new ReelController({
  reelStrips: [strip1, strip2, strip3],
  autoStop: true,
});

const results = controller.autoStopAll(winningRole);
// 全リールのStopResult[]を返却
```

## 取りこぼしと持ち越し（CarryOver）

ボーナスが内部当選したがプレイヤーがシンボルを揃えられなかった場合（取りこぼし）、`InternalLottery.setCarryOver` でボーナス当選状態を次ゲーム以降に持ち越せます。

```ts
// 全リールの停止位置決定後
if (stopResults.some((r) => r.isMiss) && winningRole.type === 'BONUS') {
  lottery.setCarryOver(winningRole);
  // プレイヤーがシンボルを揃えるまでボーナス当選状態を維持
}
```

## StopButton コンポーネント

`StopButton` はリール手動停止のUIを提供します:

```tsx
import { StopButton } from 'reeljs';

<StopButton reelIndex={0} onStop={(timing) => handleStop(0, timing)} disabled={!isSpinning} />
```
