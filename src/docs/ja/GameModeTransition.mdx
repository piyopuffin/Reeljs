# ゲームモード遷移

`GameModeManager` は4つのゲームモードを管理します: **Normal**（通常）、**Chance**（チャンス）、**Bonus**（ボーナス）、**BT**（ボーナストリガー）。

## 遷移図

    Normal --> Bonus : ボーナス当選
    Normal --> BT : forceTransition（天井等）
    Chance --> BT : WinPattern成立
    Chance --> Normal : 上限到達
    Bonus --> Normal : BIG/REG終了
    Bonus --> BT : SBB終了
    BT --> Bonus : WinPattern成立
    BT --> Normal : 上限到達

## 使用例

```tsx
import { useGameMode } from 'reeljs';

const { currentMode, currentBonusType, remainingSpins } = useGameMode({
  transitionConfig: { normalToChance: 0.02, chanceTobt: 0.3, btToSuperBigBonus: 0.1 },
  bonusConfigs: { /* ... */ },
  btConfig: { maxSpins: 50, maxPayout: 500, winPatterns: [] },
  chanceConfig: { maxSpins: 20, maxPayout: 200, winPatterns: [] },
});
```

## forceTransition（強制遷移）

`GameModeManager.forceTransition()` は確率判定・遷移バリデーションをバイパスして指定モードへ強制遷移します。天井到達時のNormal→BT直接突入やデバッグ用途に使用します。

```ts
import { GameModeManager } from 'reeljs';

const manager = new GameModeManager(config);

// 天井到達時にBTへ強制遷移
manager.forceTransition('BT');

// Bonusへ強制遷移（bonusType必須）
manager.forceTransition('Bonus', { bonusType: 'SUPER_BIG_BONUS' });
```
