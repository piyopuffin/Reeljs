# Game Mode Transition

Reeljs manages four game modes via `GameModeManager`: **Normal**, **Chance**, **Bonus**, and **BT** (Bonus Trigger).

## Transition Diagram

<Mermaid chart={`
stateDiagram-v2
    [*] --> Normal
    Normal --> Chance : Chance transition condition met
    Normal --> Bonus : Bonus win (InternalLottery)
    Normal --> BT : forceTransition (e.g. ceiling)
    Chance --> BT : WinPattern matched
    Chance --> Normal : Max spins / MaxPayout reached
    Bonus --> Normal : BIG_BONUS / REG_BONUS ended
    Bonus --> BT : SUPER_BIG_BONUS ended
    BT --> Bonus : WinPattern matched (SUPER_BIG_BONUS re-entry)
    BT --> Normal : Max spins / MaxPayout reached
`} />

## Usage

```tsx
import { useGameMode } from 'reeljs';

const { currentMode, currentBonusType, remainingSpins } = useGameMode({
  transitionConfig: { normalToChance: 0.02, chanceTobt: 0.3, btToSuperBigBonus: 0.1 },
  bonusConfigs: { /* ... */ },
  btConfig: { maxSpins: 50, maxPayout: 500, winPatterns: [] },
  chanceConfig: { maxSpins: 20, maxPayout: 200, winPatterns: [] },
});
```

## forceTransition

`GameModeManager.forceTransition()` bypasses probability checks and transition validation to force a mode change. Useful for ceiling (天井) mechanics or debug tools.

```ts
import { GameModeManager } from 'reeljs';

const manager = new GameModeManager(config);

// Force transition to BT (e.g. ceiling reached)
manager.forceTransition('BT');

// Force transition to Bonus (bonusType is required)
manager.forceTransition('Bonus', { bonusType: 'SUPER_BIG_BONUS' });
```
