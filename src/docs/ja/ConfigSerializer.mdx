# 設定シリアライズ

`ConfigSerializer` は `GameConfig` のシリアライズ・デシリアライズ・バリデーションを提供します。ラウンドトリップ特性を保証し、シリアライズ→デシリアライズで同一の設定が復元されます。

## API

### serialize

`GameConfig` をJSON文字列に変換します。

```ts
import { ConfigSerializer } from 'reeljs';

const json = ConfigSerializer.serialize(gameConfig);
// ファイル、localStorage、サーバーに保存
```

### deserialize

JSON文字列を `GameConfig` にパースします。不正なJSONや必須フィールド欠落時はエラーをスローします。

```ts
const config = ConfigSerializer.deserialize(json);
// config は完全に型付けされた GameConfig
```

### validate

値が有効な `GameConfig` かどうかを検証する型ガードです。

```ts
const data = JSON.parse(someString);
if (ConfigSerializer.validate(data)) {
  // data は GameConfig
  console.log(data.betConfig.initialCredit);
}
```

## JSON保存・読み込みパターン

保存・読み込みの完全なワークフロー:

```ts
import { ConfigSerializer } from 'reeljs';

// 保存
function saveConfig(config: GameConfig): void {
  const json = ConfigSerializer.serialize(config);
  localStorage.setItem('slotConfig', json);
}

// 読み込み
function loadConfig(): GameConfig | null {
  const json = localStorage.getItem('slotConfig');
  if (!json) return null;
  try {
    return ConfigSerializer.deserialize(json);
  } catch (e) {
    console.error('設定が不正です:', e);
    return null;
  }
}

// 使用前にバリデーション
function importConfig(data: unknown): GameConfig {
  if (!ConfigSerializer.validate(data)) {
    throw new Error('設定フォーマットが不正です');
  }
  return data;
}
```
