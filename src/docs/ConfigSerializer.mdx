# Config Serializer

`ConfigSerializer` provides serialize, deserialize, and validate operations for `GameConfig`. It guarantees round-trip fidelity — serializing then deserializing produces an identical config.

## API

### serialize

Converts a `GameConfig` to a JSON string.

```ts
import { ConfigSerializer } from 'reeljs';

const json = ConfigSerializer.serialize(gameConfig);
// Store to file, localStorage, or send to server
```

### deserialize

Parses a JSON string back to `GameConfig`. Throws on invalid JSON or missing required fields.

```ts
const config = ConfigSerializer.deserialize(json);
// config is a fully typed GameConfig
```

### validate

Type guard that checks if a value is a valid `GameConfig`.

```ts
const data = JSON.parse(someString);
if (ConfigSerializer.validate(data)) {
  // data is GameConfig
  console.log(data.betConfig.initialCredit);
}
```

## JSON Save/Load Pattern

A complete save/load workflow:

```ts
import { ConfigSerializer } from 'reeljs';

// Save
function saveConfig(config: GameConfig): void {
  const json = ConfigSerializer.serialize(config);
  localStorage.setItem('slotConfig', json);
}

// Load
function loadConfig(): GameConfig | null {
  const json = localStorage.getItem('slotConfig');
  if (!json) return null;
  try {
    return ConfigSerializer.deserialize(json);
  } catch (e) {
    console.error('Invalid config:', e);
    return null;
  }
}

// Validate before use
function importConfig(data: unknown): GameConfig {
  if (!ConfigSerializer.validate(data)) {
    throw new Error('Invalid config format');
  }
  return data;
}
```
